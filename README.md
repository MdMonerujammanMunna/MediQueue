# 🌟 **MediQueue** Tutor Booking Platform

**MediQueue** is a modern and user-friendly tutor booking web application where students can easily register, log in, browse available tutors, and book online learning sessions based on subject and time availability.  
The platform helps students manage their learning schedules efficiently by generating digital session tokens for every booking and preventing time-slot conflicts.

🚀 **Live Demo:** [Click here](https://medi-queue-psi-flame.vercel.app)

---

# 🎯 **Purpose**

The purpose of this project is to build a responsive and organized tutor booking platform that simplifies the process of finding and scheduling tutors online.  

This project demonstrates frontend and backend development skills including:

- 🔐 Authentication System
- 📱 Responsive UI Design
- 🗄️ Database Management
- ⚡ Booking Management System
- 🚫 Protected Routes
- 🔑 JWT Authentication

The system removes manual scheduling problems, avoids booking conflicts, and ensures a smooth learning experience for students.

---

# ✨ **Key Features**

- 👨‍🏫 **Browse Available Tutors**

  Students can explore available tutors with subject details, schedules, and pricing.

- 📅 **Book Tutor Sessions**

  Easily book online learning sessions based on subject and time availability.

- 🎟️ **Digital Session Tokens**

  Generates digital session tokens for every successful booking.

- 🔐 **JWT Authentication & Authorization**

  Secure authentication system using JWT Token for protected routes and user verification.

- 🔑 **Google Login Support**

  Integrated Better Auth with Google Sign-In functionality.

- 🔍 **Search & Filter Tutors**

  Search tutors by subject, category, and availability.

- 👤 **User Profile Management**

  Users can manage profile information and booked sessions easily.

- 🚫 **Protected Routes**

  Dashboard, booking pages, and profile pages are accessible only after login.

- 📱 **Responsive Design**

  Fully optimized for mobile, tablet, and desktop devices.

- 📢 **Announcement Marquee**

  Scrolling announcements section for important updates.

- 🔔 **Toast Notifications**

  Instant notifications using React Toastify.

- 🗄️ **MongoDB Database Integration**

  Securely stores tutor, booking, and user information.

- 🚪 **Logout Feature**

  Users can securely log out anytime.

- 🎨 **Clean & Modern UI**

  Smooth navigation with a simple and user-friendly interface.
